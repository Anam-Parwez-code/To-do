Project Tittle:Secure MERN Authentication
(JWT,Refresh Token,Hashed Password,Role-Based Access)
DESCRIPTION:-
This project is a production ready Secure Authentication system builtusing MERN Stack with modern security standards.It includes JWT Authentication,Refresh Token Rotation,Role-Based Access Control,and HTTp-only Secure cookies,making it suitable for real -world SaaS products,dashboards,and enterprise apps.
LIVE DEMO:
[LIVE DEMO](https://mern-authentication-ui.vercel.app/)

KEY FEATURES
1.JWT Access & Refresh Token Authentication
2.HTTP-only Secure Cookies
3.Password Hashing(bcrypt)
4.MongoDB Atlas +Compass
5.Production Deployment (vercel)
6.Modular Controllers,Middlewares,and Services
